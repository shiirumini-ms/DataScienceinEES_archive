# Repository for WWF Report

## Red Fox (*Vulpes vulpes*) population trend between 1970-2000: Impact of Rabies

This analysis is aimed at comparing the Red Fox population trend over 1970-2000, between regions affected by rabies outbreak and absent from rabies.

It is conducted as part of assessment for Data Science in Ecological and Environmental Sciences, a UG course at the University of Edinburgh.

Report can be found in `WWF Report` folder as `WWF-Report.pdf`. Report was made in R Markdown and runnable via `WWF-Report_Rmd` in the same folder.

Following signposts you to the 'Appendix' that was present in the report.

### Data

The original dataset is exracted from ['Living Planet' Database](https://www.livingplanetindex.org/search) published by **World Wildlife Fund**.

Meta data, raw, and processed data are available in `data` folder in names below:

-   LPI_data.Rdata

-   LPI_metadata_2024.pdf

-   fox.csv\* (\*only used for trial scripts available in `script` folder)

### **Plots**

Figures produced during the analyses can be found in `figure` folder in names below:

-   studysitemap.png

-   model_output.png

-   diagnostics.png

### Scripts

R scripts are available accordingly:

-   R Markdown file where cleaning and analyses are all done in one go

    `WWF Report/WWF Report.Rmd`

-   R scripts for trial runs. Data analyses were primarily done here before replicated to R Markdown.

    `script/script1_choosingspecies.R` \| `script/script2_buildmodel.R`
